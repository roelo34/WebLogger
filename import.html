Import existing case

Import a previously created csv logfile. The logfile has to have been made by WebLogger in order to import correctly.

If you want to try to import a non-standard log anyway, make sure the format is as follows:

Who Where When What Why How Result Query URL Case name

!WIP Set name for screenshots WIP!

By default, the names for the screenshots follow this format:

ProjectName_ResearcherName_CurrentDatetime.png

When setting a custom filename, you can use these options:

  • %R for researcher name
  • %D for current date
  • %T for current time
  • %P for project name
  • %U for URL
